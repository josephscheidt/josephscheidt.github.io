Current projects

My ongoing academic and personal explorations in data science

What I’m working on

With baseball season fast approaching, I am currently working on a linear regression that uses season statistics to predict MVP vote shares for each player. Using the invaluable Lahman database, through the Lahman R package, I intend to build what will hopefully be a decent prediction model, and then use that model to track the state of the MVP race in each baseball league (AL and NL) as the season progresses.

What I’m learning about

I am starting a Calculus III course, and I’m also delving deep into Python. I hope to develop some publishable Python work soon. I’m not sure when and where I’ll get to apply the advanced calculus knowledge, but there seems to be very few math concepts that don’t come in handy at some point in the world of data science, so I’m sure I will get the chance before too long.
